# 💰 Expense Tracker Pro

A full-featured **Expense Tracking App** built with the **MERN stack** (MongoDB, Express.js, React.js, Node.js). It allows users to securely register, log in, and manage their income and expenses in real-time with persistent storage and modern UI.

🔗 [Live Demo](https://expensetracker-pro.netlify.app)

---

## ✨ Features

- 🔐 Authentication (Register / Login / Logout)
- 📊 Add income and expense transactions
- 📁 View transaction history with filtering
- 📈 Dashboard with real-time stats
- 🧠 Persistent user data (MongoDB Atlas)
- 💡 Clean and responsive UI (Tailwind CSS)
- 🔐 Protected routes & JWT token-based auth
- 🌐 Deployed on Netlify (frontend) and Railway (backend)
